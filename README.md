SNITCH FASHION STORE SALES REPORT

Project overview
Snitch Fashion Store generates large volumes of sales data across multiple stores, products, and dates. However, raw transactional data does not clearly highlight business performance trends.
This project focuses on transforming raw sales data into meaningful insights using Excel-based analysis, pivot tables, and interactive dashboards to support data-driven decision-making.

Description
This project analyzes Snitch Fashion Store sales data using Excel to identify sales trends, top products, store performance, and the impact of discounts. Interactive pivot tables and dashboards are used to support data-driven business decisions.

Dependencies
Operating System: Windows 10 or above
Software: Microsoft Excel (2016 or later recommended)
