
Project Overview:
This project analyzes Snitch Store sales data using Microsoft Excel to understand revenue trends, product performance, discount impact, and store-level sales behavior. The project includes data preparation, analytical worksheets, and an interactive dashboard for business insights.

Project Objectives:
 Analyze overall sales and revenue performance
 Identify top and low performing products
 Study sales trends over time (daily & yearly)
 Evaluate the impact of discounts on revenue
 Compare revenue across products and stores
 Build an interactive Excel dashboard for insights
 
Data Pre-Processing:
 Removed duplicate and invalid records
 Checked for missing values
 Standardized column names
 Ensured correct data types (Date, Numeric, Text)
Structured data for analysis and visualization

Visualizations & Sheets:
 Dashboard – Overall KPIs and insights
 Revenue vs Product – Product contribution analysis
 Sales Trend Over Time – Time series analysis
 Revenue by Product – Product-wise revenue comparison
 Product Average Sale – Average sales per product
 Discount vs Revenue – Discount impact analysis
 Product Sold Over Years – Yearly product trends
 
Insights:
 Certain products consistently generate higher revenue
 Discounts influence revenue differently across products
 Clear seasonal and daily sales patterns observed
 
Conclusion:
 This Excel-based analysis provides actionable insights into sales performance, product behavior, and discount effectiveness. The dashboard enables quick decision-making for business stakeholders and highlights    areas for optimization.
 
File Structure
 Snitch Sale Main / Snitch Sale – Raw & cleaned sales data
 Product Code – Product dimension table
 Store Code – Store dimension table
 Analysis Sheets – Pivot-based insights
 Dashboard – Final interactive summary
